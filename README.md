# 余白 yohaku

> *yohaku (余白)* — Japanese for "white space" or "margin"

A minimalist text editor that lives entirely in your browser and stores everything in the URL hash.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + S` | Save to URL |
| `Ctrl + Q` | Show QR code |
| `Ctrl + L` | Copy link |
| `Tab` | Insert tab character |


## Why "余白"?

余白 (yohaku) represents the intentional empty space in Japanese art and design. It's the breathing room that gives meaning to what's present. This editor embraces that philosophy — a blank canvas that holds only what matters.
